## [3.4] - 2026-03-11

### ✨ Новые возможности v3.4

#### Bluetooth LE Tools (опция 16)

**📡 Bluetooth LE (Low Energy) Tools**:
- **Основные BLE инструменты:** gatttool, hcitool, btmgmt, bluez-deprecated
- **BLE сканеры и анализаторы:** bleah, gattlib, pygatt
- **Python BLE библиотеки:** bluepy, bleak (кроссплатформенный), bleson
- **Специализированные инструменты:** nrfutil (Nordic SDK), microbit (BBC micro:bit)
- **Утилиты:** Espruino IDE

**🔧 Возможности:**
- Сканирование BLE устройств (hcitool lescan)
- Подключение к GATT серверам
- Чтение/запись характеристик
- Анализ BLE рекламы
- Работа с Nordic nRF устройствами
- Программирование BBC micro:bit
- Кроссплатформенная разработка (bleak)

**📝 Примечание:** Требуется root и BLE адаптер

**📖 Основные команды:**
```bash
gatttool -I           # Интерактивный режим
hcitool lescan        # Сканирование BLE устройств
btmgmt info           # Информация о Bluetooth адаптере
```

## [3.0] - 2026-03-11

### ✨ Новые возможности v3.0

#### Восстановление и Backup
- **Восстановление из backup** — функция отката изменений
  - `restore_from_backup()` — восстановление пакетов из backup
  - `list_backups()` — просмотр и выбор backup для восстановления
  - Опция 4 в настройках — 🔄 Восстановление из backup
  - Восстановление pkg, pip пакетов и конфигов

#### Профили установки
- **Профили установки** — предустановленные конфигурации
  - `install_profile()` — установка профиля
  - `show_profiles_menu()` — меню выбора профилей
  - Опция P в главном меню — 📦 Профили установки
  - **Minimal** (~50MB) — git, wget, curl, nano, vim, python
  - **Standard** (~200MB) — + ruby, golang, nodejs, figlet, ssh
  - **Full** (~500MB) — + clang, php, mariadb, metasploit, nmap
  - **Pentest** (~800MB) — + aircrack-ng, hashcat, wifite, ghidra

#### Проверка совместимости
- **Комплексная проверка системы**:
  - `check_termux_version()` — проверка версии Termux
  - `check_android_version()` — проверка версии Android и SDK
  - `check_root_status()` — проверка root-прав
  - `check_compatibility()` — общая проверка совместимости
  - Проверка: Termux, Android (SDK >= 21), root, место, интернет

#### Авто-обновления
- **Авто-обновление скрипта**:
  - `check_for_updates()` — проверка доступных обновлений
  - `install_update()` — установка обновлений
  - Опция U в главном меню — 🔄 Проверить обновления
  - Загрузка с GitHub, backup, авто-перезапуск
