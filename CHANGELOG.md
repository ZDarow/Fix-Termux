## [2.2] - 2026-03-11

### ✨ Новые возможности
- **Подтверждение перед установкой** — запрос y/N перед началом установки
- **Статистика установки** — подсчёт пакетов, ошибок, репозиториев
- **Тестовый режим (Dry Run)** — симуляция установки без реальных изменений
- **Авто-подтверждение** — режим для автоматической установки без запросов
- **Итоговая статистика** — показ статистики при выходе из программы

### 🔧 Исправления
- Обновлены функции `install_package()`, `clone_repo()`, `run_command()`
- Добавлен подсчёт статистики в реальном времени
- Добавлена поддержка dry-run режима

### ⚙️  Настройки
- **S** — меню настроек (Dry Run, Auto Confirm)
- **T** — просмотр статистики

### 📊 Статистика
- Установлено пакетов
- Ошибок установки
- Склонировано репозиториев
- Выполнено команд
- Процент успешной установки

## [2.0] - 2026-03-11

### ✨ Новые возможности
- **Рефакторинг кода** — весь код разбит на модульные функции
- **Логирование** — все действия записываются в `$HOME/fix-termux.log`
- **Проверка системы** — автоматическая проверка интернет-соединения и места на диске
- **Цветное меню** — улучшенный интерфейс с иконками и описаниями
- **Возврат в меню** — после завершения установки можно продолжить работу
- **Обработка ошибок** — корректная обработка ошибок установки

### 🔧 Исправления
- Исправлена опечатка `$gteen` → `$green`
- Заменены `apt install` → `pkg install` для Termux
- Удалены устаревшие команды `pip2`
- Исправлено дублирование `git clone git clone`
- Исправлена опечатка `inatall` → `install`
- Исправлена переменная `$ HOME` → `$HOME`
- Удалён несуществующий пакет `gem install bundle`

### 📦 Улучшения установки
- Добавлены флаги `-y` для автоматической установки
- Добавлена проверка ошибок для `git clone`
- Унифицированы команды (`pkg` для Termux, `apt` для Kali)
- Удалены несуществующие пакеты
- Реализована опция 5 (Ngrok) — загрузка и установка
- Добавлены комментарии к функциям

### 📚 Документация
- Обновлён README.md с правильной структурой
- Создан QWEN.md с полной документацией
- Создан CHANGELOG.md
