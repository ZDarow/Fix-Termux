# 🚀 Fix-Termux Launcher

**Интуитивное древовидное меню для запуска установленных инструментов**

---

## 📋 Обзор

**Launcher** — это интерактивное меню для быстрого запуска всех установленных через Fix-Termux инструментов. Аналогично установочному меню, но для **запуска**, а не установки.

---

## 🎯 Возможности

- ✅ **Древовидное меню** — 8 разделов инструментов
- ✅ **Проверка установки** — показывает ✅/❌ для каждого инструмента
- ✅ **Интерактивный запуск** — запрос параметров (IP, URL, файлы)
- ✅ **Возврат в меню** — после выполнения инструмента
- ✅ **Горячие клавиши** — `B` (назад), `0` (выход)
- ✅ **Цветной интерфейс** — красивое оформление с иконками

## 🚀 Быстрый Старт

### Способ 1: Прямой запуск

```bash
cd ~/Fix-Termux
chmod +x launcher.sh
bash launcher.sh
```

### Способ 2: Через главное меню Fix-Termux

```bash
bash Fix-Termux.sh
# Выбрать: L (Запустить Launcher)
```

## ⌨️ Горячие Клавиши

| Клавиша | Действие |
|---------|----------|
| `1-8` | Выбор раздела |
| `1-N` | Выбор инструмента |
| `B` | Назад в главное меню |
| `0` | Выход из Launcher |
| `Enter` | Продолжить после выполнения |

## 🔧 Настройка

### Добавление своих инструментов

Откройте `launcher.sh` и найдите нужную функцию меню, например `show_dev_menu()`:

```bash
show_dev_menu() {
    show_header
    
    echo -e "  ${CYAN}📱 ТЕРМУКС И РАЗРАБОТКА${reset}"
    echo ""
    echo -e "  $(check_installed python3) ${GREEN}1${reset}) Python 3"
    echo -e "  $(check_installed YOUR_TOOL) ${GREEN}N${reset}) Your Tool"  # Добавить
    ...
```

Добавьте обработку в `case`:

```bash
case $choice in
    ...
    N) run_tool "your_tool" ;;  # Добавить
    ...
```

## ⚠️ Примечания

1. **Проверка установки** — если инструмент помечен `❌`, установите его через Fix-Termux
2. **Root права** — некоторые инструменты (Nmap, Metasploit) требуют root
3. **Bluetooth** — требует Bluetooth адаптер и поддержку в Android
4. **GUI инструменты** — Wireshark, Qt Bluetooth требуют X11/GUI
